# 🚀 Aurelia – AI-Powered Luxury Travel Consultant

## Overview

Aurelia is an AI-powered Luxury Travel Consultant designed using advanced Prompt Engineering techniques. The project demonstrates how a Large Language Model (LLM) can be transformed into a professional, secure, and compliant customer-facing assistant through System Prompting, Persona Engineering, Few-Shot Learning, and AI Guardrails.

The system maintains a consistent luxury travel consultant persona while defending against prompt injection attacks, competitor manipulation, prompt harvesting attempts, discount abuse, and other adversarial inputs.

---

## 🎯 Project Objective

The primary goal of this project is to demonstrate:

* System Prompt Design
* Persona Engineering
* Few-Shot Prompting
* AI Safety Guardrails
* Compliance Validation
* Adversarial Stress Testing

Rather than functioning as a generic chatbot, Aurelia operates as a specialized Luxury Travel Consultant with clearly defined behavior, constraints, and knowledge boundaries.
